# is_palindrome 

Checks if a string is a palindrome

## Installation

Add this line to your application's Gemfile:

    gem 'palindrome'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install palindrome

## Usage

Run
 
    $ palindrome <string>

to check if the string is a palindrome. (Returns true or false)
